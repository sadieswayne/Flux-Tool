## ğŸ“„ Getting Started

### 1. Clone the Repo

```bash
git clone https://github.com/yourusername/flux-tool.git
cd flux-tool

# Frontend
cd frontend
npm install

# Backend
cd ../backend
pip install -r requirements.txt
```

### 3. Environment Variables

Create a `.env` file in both `frontend` and `backend` folders.  
Refer to the provided `.env.example` file for the required environment variables.

---

### 4. Run Locally

```bash
# Backend
cd backend
uvicorn main:app --reload

# Frontend
cd ../frontend
npm run dev
```
